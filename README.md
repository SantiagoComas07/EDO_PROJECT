# EDO_PROJECT
    This project is about implementing a formula to manage the progress of teams considering  different factors related to teamwork.

## Features
- *Using EDO* – Implements a formula to manage and analyze team progress based on different factors.
- Agile Methodology
- research project

## Technologies
- HTML3
- CSS3
- Javascript ES6
- Tailwind 4.1.16 (framework)
- chart.js 4.5.1 (Librery)
- Vite 7.1.7

## Run project

```bash
 npm run dev
```
